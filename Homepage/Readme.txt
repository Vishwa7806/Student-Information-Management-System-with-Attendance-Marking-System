After extracting the zip file, place the Homepage folder in the following location in your system
Location - "C:\xampp\htdocs"

Database - 
Step 1: Go to localhost/phpmyadmin
Step 2: Create a database named as "attendance"
Step 3: Go to import section and select the choose file option
Step 4: Choose the "attendance.sql" file from the folder name "sql" inside the homepage folder
Step 5: Click the import button
